Operating Notes

Select a set of parameters for random values. Pick new random values for individual parameters or all at once.

  • App can be installed as an app on your device and will work offline. There should be an install option in your web browser.
  • The "space" key will start stop any media playing.
  • The "P" key will pick a new set of parameters.
  • The media player needs manual interaction to start playing but later changes will play automatically.
