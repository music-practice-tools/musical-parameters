About Musical Parameters

A web app that displays sets of parameters each with several possible values. The displayed value for each parameter can be randomly selected. Designed for selecting musical parameters for practicing etc. but should be generally useful.

It is possible to select new values for all parameters or just a single one. Audio media can be played for a selected value.

The available parameters are defined in simple configuration files. In addition values for the media file URL can be passed in the URL query string.

Operating Notes

  • Select a set of parameters in top row with 🔄 icon button at the right of the row.
  • Pick new random values for individual parameters using 🔄 icon buttons on the right end of each row.
  • Clicking on a parameter value will present a list of all values so one can be selected.
  • The "N" key will pick a new value for all parameters.
  • The number keys will pick a new value for the parameter at the row specified (first is 1).
  • The Space and "P" keys will play / pause media playing if available. As will clicking/touching background areas.
  • Media playing is initially paused until a manual interaction occurs.
  • App can be installed as an app on your device and will work offline. There should be an install option in your web browser.
