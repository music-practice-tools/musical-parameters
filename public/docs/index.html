About

A small web app that displays sets of parameters each with several possible values. Designed for selecting musical parameters for practicing etc. but should be generally useful.

Select new values for all or a single parameter. Supports audio media playing of a file for a value. The sets of parameters are defined in config files. Parameters can also be passed in the URL query string for use in the config file.

Operating Notes

  • Select a set of parameters in top row.
  • Pick new random values for individual parameters or all at once using icons on the right end of each row.
  • Clicking on a parameter value will present a list of all values so one can be selected.
  • If a parameters has extra values then you select which to display: values, both, extra.
  • The "N" key will pick a new set of all parameters.
  • The "P" key will start / stop media playing if available. As will clicking/touching background areas.
  • Media playing is initially paused until a manual interaction occurs.
  • App can be installed as an app on your device and will work offline. There should be an install option in your web browser.
