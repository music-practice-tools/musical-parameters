About Musical Parameters

A mobile friendly web app that presents sets of parameters each with a current value selected and shown from several possibilities. The displayed value for each parameter can be randomly selected, both individually or all at once. In addition, the displayed value for each can be manually selected.

Additional features alow the provision of audio files to be played for selected values and also the display of a text note based on the current values.

This is not a ready-to-use app, rather like a word processor, it is a tool that requires configuration files (ie documents) that specify a set of the parameters, values, media files and text notes that are presented when using the app. These configuration files are in yaml format which aims to be human friendly but is still a technical language. We provide examples below. In addition a value for the base URL of any media file can be passed in the URL query string.

Originally designed to generate random musical parameters for practicing, composing etc. in response to a request by Al Swainger. However, it should be generally useful.

Operating Notes

  • Select a set of parameters in top row with 🔄 icon button at the right of the row.
  • Pick new random values for individual parameters using 🔄 icon buttons on the right end of each row.
  • Clicking on a parameter value will present a list of all values so one can be selected.
  • Clicking ⋮ on a row opens the settings. Clicking X or the Esc key will close.
    • Any: pick any value from all values each time.
    • Every: pick every single value until all have been chosen.
    • Lock: do not change the value when pick all clicked.
  • When media files are provided as part of the configuration:
    • Playing is initially paused until a manual interaction occurs; this is due to browser behaviour.
    • At the end of playing the same track can repeat (Loop 1, the default) or new values chosen (Shuffle).
    • Playback speed can be varied from "Normal" to be faster or slower with pitches kept the same.
  • The "N" key will pick a new value for all parameters.
  • The number keys will pick a new value for the parameter at the row specified (first is 1).
  • The Space and "P" keys will play / pause media playing if available. As will clicking/touching background areas.
